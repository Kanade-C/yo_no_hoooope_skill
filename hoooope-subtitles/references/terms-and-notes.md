# HOOOOPE Terms And Notes Router

This compatibility reference only routes topic-specific material. Load the smallest reference set needed for the current stage; bundled scripts load these split references directly.

- `references/terms-glossary.md`: fixed HOOOOPE terms, names, brand traps, and common bad-term corrections.
- `references/deepseek-prompts.md`: DeepSeek V4-Pro translation, self-polish, and note-generation prompt policy.
- `references/note-style.md`: Chinese viewing-note style, length, compression, topic selection, and summary output shape.
- `references/review-policy.md`: Codex final review policy, public-release strict coverage, reduced-review mode, QA gates, and tone priority.

For translation and final fixed-term checks, load `terms-glossary.md`, `deepseek-prompts.md`, and `review-policy.md`.

For summary editing, load `terms-glossary.md`, `note-style.md`, and `review-policy.md`.
